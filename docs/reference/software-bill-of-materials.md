# Software bill of materials

An SBOM is a structured list of components, libraries, and modules required to build a piece of software, including their supply chain relationships.

SBOMs are required under the EU Cyber Resilience Act and referenced by the US NTIA as a key element of software transparency.

SBOMs for each Ubuntu Core release are available from the following links:

- [Ubuntu Core 24 SBOM](sboms/core24_1587.spdx2.3.json)
- [Ubuntu Core 22 SBOM](sboms/core22_2411.spdx2.3.json)
- [Ubuntu Core 20 SBOM](sboms/core20_2769.spdx2.3.json)
